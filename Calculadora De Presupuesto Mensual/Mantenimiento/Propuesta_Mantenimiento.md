La calculadora del presupuesto mensual permite registrar ingresos y gastos, generando un balance general para evaluar el estado financiero del usuario.

# Fundamentación teórica: Mantenimiento de Software

## Definición general
El mantenimiento de software abarca actividades posteriores a la entrega del sistema para corregir errores, mejorar desempeño o adaptarlo a nuevos entornos.

---

## Tipos de mantenimiento

| Tipo | Descripción | Ejemplo |
|------|-------------|---------|
| Correctivo | Corrige errores del sistema | Ajustar fórmula del balance |
| Adaptativo | Ajusta el sistema a nuevos entornos | Actualizar para versión nueva de SO |
| Perfectivo | Mejora funciones sin fallos previos | Agregar gráfica de gastos |
| Preventivo | Previene problemas futuros | Revisar base de datos |

---

## Costos del mantenimiento
Incluyen:  
- Corrección de errores  
- Actualizaciones por cambios en el entorno  
- Nuevas funcionalidades  
- Pruebas de regresión  

---

## Etapas del mantenimiento

| Etapa | Descripción |
|-------|-------------|
| Análisis | Identificación del problema |
| Evaluación | Determinar impacto |
| Modificación | Implementación técnica |
| Pruebas | Validación del cambio |
| Documentación | Registro y entrega |

# Análisis del caso

## Problema 1: Errores en el cálculo del balance  
**Tipo:** Correctivo  
**Solución:** Validación de datos correctos y completos.

---

## Problema 2: No hay almacenamiento permanente  
**Tipo:** Preventivo  
**Solución:** Implementar base de datos local o JSON persistente.

---

# Área de mejora  
**Tipo:** Adaptativo  
**Solución:** Crear versión PWA multiplataforma.

---

# Cambios funcionales

## 1. Validación de datos  
Evita datos inválidos antes del cálculo.

## 2. Almacenamiento persistente  
Base de datos local con CRUD.

## 3. Adaptación PWA  
Responsive, offline y compatible con navegadores.

---

# Reflexión final
El mantenimiento garantiza evolución, estabilidad, calidad y adaptabilidad del software, asegurando su utilidad a largo plazo.
